Classification:
Submission Format
Prediction File
Submit a CSV file named prediction_task_a.csv with the following columns:

id: Unique identifier of the text item
label: One of the 20 predefined DNK classification labels as integers
Code Folder (Evaluation Phase only)
Create a folder named code
Include your main.py file in this folder
Ensure your code can reproduce your predictions
Submission Package
Place all required files in a zip folder
The zip folder can have any name
Ensure the internal structure matches the requirements above
